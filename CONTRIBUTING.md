
# Contributing to Dan

First off, thank you for your interest in **Dan**! 

## Current Project Status
**Dan** is currently a personal project maintained solely by its creator. To ensure the core architecture (specifically the rendering engine and the "Rope" data structure) remains cohesive during this early phase, **we are not accepting pull requests or active code contributors at this time.**

This isn't because we don't value your help, it's simply to allow the project's vision to stabilize before opening the floor to external changes.

---

## How You Can Help
Even though the codebase is currently "read-only" for the public, your input is incredibly valuable in other ways:

### 1. Reporting Problems (Issues)
If you find a bug, a crash, or a strange rendering artifact, please let us know! 
* **Check existing issues** to see if it has already been reported.
* **Open a new Issue** with clear steps to reproduce the bug.
* Please include your terminal emulator (e.g., Alacritty, iTerm2, Windows Terminal) and your OS, as terminal escape sequences can behave differently across environments.

### 2. Sharing Ideas (Discussions)
Have a thought on a better shortcut for "Select All"? Or an idea for a plugin interface?
* Head over to the **Discussions** tab.
* This is the best place for feature requests, "how-to" questions, or just showing off your `config.toml` setup.

---

## Future Roadmap
If you have a major feature in mind, feel free to discuss it in the Discussions area so it can be considered for the official roadmap. Thank you for using **Dan**!
